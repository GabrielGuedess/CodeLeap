## Getting Started

### Prerequisites

_NodeJs_

### Installation

1. Clone the repository.

   ```sh
   git clone https://github.com/GabrielGuedess/CodeLeap
   ```

2. Install the dependencies.

   ```sh
   # With npm
   npm i

   # With yarn
   yarn
   ```

3. Start project.

   ```sh
   # With npm
   npm run dev

   # With yarn
   yarn dev
   ```
